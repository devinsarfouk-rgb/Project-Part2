How It Works
1. Feature Extraction
Extracts 30 URL features or 18 email features (domain info, URL structure, language patterns, etc.)
2. ML Ensemble
Random Forest + Gradient Boosting + Logistic Regression vote on the probability of phishing.
3. Heuristic Scoring
Rule-based scoring adds explainability and catches patterns the ML may miss.
4. Educational Output
Each flag explains the phishing technique, helping you recognise it in future.
